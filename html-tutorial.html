Tutorial HTML

Você está em: Site do Luisão WebMaster >> Tutoriais >> Tutorial HTML

Bem, aqui estou como "colunista" na Programação On-Line. Aqui eu estou apresentando um Tutorial de HTML, a linguagem das homepages. Criar uma homepage é mais fácil do que parece, e com as coisas mais ou menos "mastigadas" aqui, é ridiculamente simples. Aqui eu não vou abordar somente os aspectos mais técnicos, mas também alguns um pouco mais subjetivos. As primeiras perguntas que eu lhe faço são: sobre o que é a página que você quer fazer? O que vai ter nela? Se as respostas não estão claras, é melhor dar um tempo, e depois começar. Uma boa idéia é ir preparando os textos (assumindo que uma homepage feita em HTML é constituída basicamente de texto), antes de se preocupar com as formatações para a Web. Isso lhe poupará trabalho, eu garanto. Preparei a estrutura do curso em dez módulos, ou "lições", que serão abertas, a princípio, semanalmente!

Tags básicas

Então você quer fazer uma página? É fácil. Você só precisa aprender alguns truques. É claro que existem esses programas próprios fazer páginas, como o FrontPage (o próximo Word também). Com ele você pode criar uma página do mesmo modo que cria um documento. Se você quer se aprofundar, o que é ótimo já que você terá um controle maior sobre a sua página, eu recomendo aprender HTML, a linguagem das homepages. É muito fácil, já que nem é uma linguagem de programação, é uma linguagem de formatação de texto (markup). E você não precisa de nenhum software especial, você pode fazê-lo até no Bloco de Notas! Por exemplo, para escrever "My name is Luis Carlos", com as palavras "Luis Carlos" em negrito, você escreveria: My name is <b> Luis Carlos</b>

Os sinais <> são chamados "tags", e eles são os comandos desta linguagem. Algumas tags úteis são: <b> para negrito (bold); <i> para itálico; <u> para sublinhado (underline); etc...

Para fazer um link, use uma tag <a>, com o argumento href, assim: <a href="http://terra.com.br">aqui</a> a palavra "aqui" vai aparecer azul e sublinhada (a aparência padrão de um link).

Outra tag útil é a <font>, que tem um monte de opções, como: <font size="-2" color=red>Texto pequeno e vermelho</font>
Como você provavelmente já percebeu, você não precisa especificar as opções na tag de encerramento (aquela com a barra /, como em <a> e </FONT>).
